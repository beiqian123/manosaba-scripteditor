# 使用帮助

> 更新日期：2026-06-09

欢迎使用剧本编辑器！这是一个专为视觉小说游戏设计的剧本创作工具。

刚接触的话不用担心，跟着下面的介绍一步步来，你很快就能上手。


## 界面概览

打开编辑器后，整个界面分为以下几个区域：

- **顶部菜单栏**：文件、编辑、工具、设置、帮助
- **顶部搜索框**：搜索剧本内容和工具
- **左侧侧边栏**：快捷工具列表，按面板和分类组织
- **中央编辑区**：写剧本的地方，支持纯文本编辑、脚本解析、阅读模式三种视图
- **底部信息栏**：版权信息

**一句话流程**：在编辑区写剧本 → 用侧边栏快捷工具辅助 → 导出剧本文件。


---

## 写剧本的基本规则

剧本由一行行的文本组成，每一行可以是：

| 类型 | 写法 | 示例 |
|------|------|------|
| **命令** | 以 `@` 开头 | `@print "你好"` — 角色说一句话 |
| **对话** | 角色名 + 英文冒号 + 内容 | `希罗: 我一定会找到真相的` |
| **旁白** | 直接写的描述文字 | `雨夜，空无一人的小巷` |
| **注释** | 以 `;` 开头，仅供自己看 | `; 这段要改一下` |

编辑器会自动给不同类型上色：命令紫色、对话绿色、数字橙色、注释灰色，一目了然。

## 三种编辑模式

编辑区顶部有三个标签页，可以随时切换：

### 纯文本编辑（默认）

标准的代码编辑器，直接用语法写剧本。支持：
- **语法高亮**：命令、参数、字符串、数字等自动着色
- **自动补全**：输入 `@` 弹出命令列表，上下键选择，Enter 确认
- **快速输入**：输入已定义的词条前缀时自动弹出补全
- **行号显示**：左侧显示行号，方便定位

### 脚本解析

将剧本逐行解析为结构化的卡片视图：

- 每行显示类型标签（命令/对话/旁白/注释）
- 命令名和参数语法高亮
- 支持**拖拽排序**：直接拖动卡片调整顺序
- 支持**悬停编辑**：鼠标移到卡片上显示编辑/删除按钮
- 支持**从侧边栏拖入**工具

### 阅读模式

想单纯看剧情效果，不想被命令代码干扰？用阅读模式。

点击「设置 → 阅读模式」，有两种级别：

- **显示简略信息**：完全隐藏命令，只显示角色对话和旁白，像在读小说
- **显示详细信息**：在命令位置显示 `[命令名 参数]` 的简要标记

阅读模式下对话用引号包裹、角色名突出显示，体验更好。

## 写剧本时的辅助功能

### 自动补全（代码提示）

在编辑区输入 `@` 后，会自动弹出一个列表显示所有可用命令。上下键选择，Tab 或 Enter 确认。不需要的话可以去「工具 → 代码提示」关掉。

### 快速输入

就像手机输入法的常用语，可以预先存一些常用词条，写剧本时快速插入。

**普通列表**：存一些常用的名字或短语。比如存了「希罗」「Ema」，在编辑区打出「希」字就会出现补全提示。

**分类快速输入**：把词条分组，每组一个**大写字母**作为触发前缀：

| 前缀 | 分组 | 说明 |
|------|------|------|
| `R` | 人物 | Role 的首字母 |
| `D` | 地名 | Di / 地方的首字母 |
| `S` | 术语 | 术语 |
| `W` | 物品 | 物品 |

在编辑区输入 `R` 弹人物列表、输入 `D` 弹地名列表。还支持**拼音首字母筛选**，比如输入 `Rxl` 会筛出「希罗」相关的项。

**管理方法**：点击「工具 → 快速输入列表」，里面有两个标签页：

1. **普通列表** — 多行文本框，一行一个词条
2. **分类快速输入** — 可视化分组管理，每个分组可展开查看和编辑词条

弹窗底部有「导入 CSV」「导出 CSV」按钮，方便备份和分享。

> 第一次使用时，如果本地没有数据，编辑器会自动从 `dev_初始配置` 文件夹加载默认词条。

### 命令悬浮提示

鼠标停在 `@命令` 所在行上，会弹出提示框，显示：
- 命令名称和所属分类
- 输出的完整模板代码
- 参数列表（必填/可选）
- 如果有多个重载版本，可切换查看

提示框可以**固定**住（默认按 F2），固定后不会自动消失，方便边看边写。固定后还会多出「编辑」按钮，点击后直接在编辑器里修改当前命令的参数。

### 侧边栏快捷工具

左侧侧边栏里有很多预设好的快捷工具按钮，按功能分成了不同的面板和分类：

- **日常模式**：对话、表演、背景、转场、音频、UI 控制、剧本跳转、分支选项、证物等
- **高级功能**：角色动画、运镜控制、运镜模板、庭审控制、魔女图鉴、系统控制等

**基本用法**：
- 点击工具按钮→如果弹出填写参数的窗口，填好确认后自动插入对应命令
- **Shift + 点击**：不弹窗，直接插入带默认值的模板，快速编写
- 分类标题可点击**折叠/展开**
- 侧边栏顶部下拉菜单可切换面板

### 快捷键

如果工具有配置快捷键（如 `Ctrl+O`），在编辑区按下即可快速插入对应命令。快捷键可在配置编辑器中查看和修改。

### 搜索功能

顶部搜索框功能强大，支持两种搜索：

1. **搜索剧本内容**：输入关键词按 Enter 搜索剧本中的文本，◀ ▶ 切换匹配项
2. **搜索工具命令**：输入关键词自动弹出下拉面板，显示匹配的工具名和分类名，支持拼音首字母搜索。输入 `@xxx` 可直接搜索命令输出。上下键选择，Enter 直接插入

### 信息面板（info.json）

侧边栏除了快捷工具外，还有一个「信息面板」用于管理游戏的 info.json 数据：

- 角色、线索、证物、场景等信息集中管理
- 可查看、编辑、新增、删除各项数据
- 支持多语言条目管理
- 与快捷工具联动，下拉选项自动从 info.json 数据生成

## 错误检查功能详解

点击「编辑 → 错误检查」，逐行扫描剧本，检查以下问题：

| 检查项 | 说明 |
|--------|------|
| 命令格式错误 | `@` 后面没有命令名 |
| 未定义的命令 | 使用了配置中没有的命令（警告） |
| 未知参数 | 命令后面跟了不认识的参数名 |
| 中文冒号 | 参数值处用了中文冒号 `：` 而不是英文冒号 `:` |
| 中文弯引号 | 命令行中出现了中文弯引号 `""` 而不是英文直引号 `""` |
| 对话多冒号 | 一行对话中出现多个冒号 |
| 多余分隔符 | 对话或旁白中出现了 `\|` 或 `&` 符号 |

检查结果在弹窗中展示，每条标注**行号:列号**，错误排前、警告排后，顶部有统计卡片。关闭弹窗后，有问题的行左侧会显示红/橙色标记。

## 配置编辑器（完整指南）

点击「设置 → 配置编辑器」，进入快捷工具的详细管理页面。这里是整个编辑器的核心自定义区域。

### 面板管理

顶部的「当前面板」下拉框选择要编辑的面板。点击旁边的「管理」按钮可以：

- **添加面板**：创建新的面板（如「自定义面板」）
- **编辑面板**：修改面板名称，勾选该面板要显示哪些分类
- **删除面板**：删除不需要的面板

> 面板的「包含分类」留空 = 显示所有分类。

### 分类管理

左侧分类列表区域：

- **添加分类**：点击「+ 添加分类」，输入分类名称即可
- **编辑分类**：鼠标移到分类上点击 ✏️ 编辑名称
- **删除分类**：点击 🗑️ 删除，会同时删除该分类下的所有工具
- **拖拽排序**：拖动分类标题可调整顺序
- **点击选中**：点击分类标题可在右侧查看该分类下的所有工具

### 工具管理

每个分类下的工具列表：

- **添加工具**：点击分类下的「+ 添加选项」
- **编辑工具**：点击工具名称，在右侧编辑面板修改
- **删除工具**：点击 🗑️ 删除
- **拖拽排序**：拖动 `⋮⋮` 可调整顺序
- **跨分类拖拽**：拖到其他分类上可移动过去

工具右侧显示类型标签：「弹窗」或「直接」。

### 编辑工具 — 基本信息

点击工具后，右侧编辑面板展示详细信息：

| 配置项 | 说明 |
|-------|------|
| **选项名称** | 工具在侧边栏显示的名称 |
| **图标（Emoji）** | 显示在名称前面的图标，如 💬 🎭 🔊 |
| **类型** | 弹窗输入 / 直接插入 |
| **快捷键** | 支持 Ctrl/Alt/Shift + 字母，如 `Ctrl+O` |
| **悬浮提示** | 开关控制鼠标悬停在该命令上时是否显示提示窗 |

### 编辑工具 — 直接插入类型

直接插入类型只需配置「输出内容」，点击工具后会将此内容直接插入到编辑器光标位置。

### 编辑工具 — 弹窗输入类型

弹窗类型需要配置完整的弹窗表单：

**窗口标题**：弹窗顶部的标题文字。

**描述说明**：支持丰富格式——可以加粗、下划线、插入图片、音频、超链接，使用顶部的富文本工具栏：

| 按钮 | 功能 |
|------|------|
| **B** | 加粗选中文字 |
| **U** | 下划线 |
| 🖼️ | 插入图片链接 |
| 🔊 | 插入音频链接（点击可预览播放） |
| 🔗 | 插入超链接 |

**输出模板**：使用 `{1}` `{2}` `{3}` 等占位符，对应表单字段的顺序。例如：
```
@print "{1}" Author:{2} WaitInput:true Wait:true
```

模板下方会自动显示每个占位符对应的字段名称的提示。

### 字段管理

在弹窗配置的「表单字段」区域，点击「+ 添加字段」添加新的输入项。

**字段名称**：表单中显示的名称。

**字段类型**：支持以下类型：

| 类型 | 说明 |
|------|------|
| 文本输入 | 普通文本框 |
| 数字输入 | 数字输入框，可设置步进值 |
| 日期选择 | 日期选择器 |
| 下拉选择 | 下拉菜单，固定选项 |
| 可编辑下拉 | 下拉菜单 + 可手动输入 |
| 复选框 | 勾选框，可分别设置选中/未选中的值 |
| 图片路径 | 图片路径输入 |
| 音频路径 | 音频路径输入 |
| 条目列表 | 可自由增删条目的列表（如分支选项列表） |

**条目列表字段**（item_list）是一种特殊的字段类型，用于管理多条结构相同的数据（如多个选项按钮）。配置时需定义：
- **子字段**：每个条目内的输入项，支持文本、数字、下拉等类型
- **条目模板**：每个条目输出的命令模板，使用 `{1}` `{2}` 对应子字段顺序
- **块前缀/后缀**：整个列表块开头和结尾的固定命令

**占位提示**：输入框中显示的灰色提示文字。

**下拉选项（下拉类型专用）**：用中文顿号「、」分隔多个选项。支持 `显示文本|实际值` 格式：
```
艾玛（Ema）|Ema、希罗（Hiro）|Hiro、诺亚（Noah）|Noah
```
下拉菜单显示「艾玛（Ema）」，但插入脚本时使用「Ema」。

**字段属性**：

| 属性 | 说明 |
|------|------|
| 表单必填 | 勾选后用户必须填写此项才能插入，字段名旁显示红色 `*` |
| 脚本分类 | 「脚本必要字段」参与主输出模板 `{N}` 替换；「脚本可选字段」需独立输出模板，仅在填写时追加 |
| 选填输出模板（可选字段专用） | 如 `time:{N}`，用户填写 0.5 时输出 `time:0.5` |
| 默认值 | 打开工具时该字段自动填入此值。复选框填 `checked` 默认选中 |

**管理字段**：每个字段右侧有编辑和删除按钮，可随时修改。

### 保存和下载

编辑完所有配置后，点击编辑面板底部的「保存修改」保存当前工具的改动。

右上角的「下载配置」按钮可以将整个配置保存为 JSON 文件，方便备份。

### 返回编辑器

点击左上角的「← 返回编辑器」回到主编辑界面。

## 常见问题

### Q: 我写的剧本存在哪里？刷新会丢失吗？
A: 编辑器自动保存到浏览器本地存储，刷新不会丢。但建议定期导出到电脑上，以防误操作或清除浏览器数据。

### Q: 如何备份我的快捷工具配置？
A: 两种方式：
1. **CSV 备份**（仅快速输入数据）：在「快速输入列表」弹窗中点击「导出 CSV」
2. **JSON 完整备份**（所有工具配置）：在「文件 → 导出自定义选项」或配置编辑器中「下载配置」

### Q: 如何导入别人分享的配置？
A: CSV 文件在「快速输入列表」弹窗中点击「导入 CSV」；JSON 备份从「文件 → 导入自定义选项」导入。

### Q: 不小心把数据搞乱了怎么办？
A: 点击「工具 → 加载为默认数据」，所有数据恢复到最初的状态。注意会覆盖当前数据，建议先导出备份。

### Q: 为什么有些命令不显示悬浮提示？
A: 可能原因：1) 该工具在配置编辑器中关闭了悬浮提示开关；2) 配置文件中没有定义这个命令。

### Q: 错误检查标记怎么清除？
A: 不需要手动清除——修改编辑器里的任何内容（打字、删除、粘贴等），所有标记会自动消失。

### Q: 脚本解析模式下的拖拽排序怎么用？
A: 直接拖动任意卡片到目标位置即可。也可以从左侧侧边栏拖拽工具到解析区的指定位置插入新行。

### Q: 条目列表是什么？怎么用？
A: 条目列表是弹窗工具中的一种特殊字段，用于管理多条结构相同的数据。比如在分支选项工具中，可以自由添加/删除选项按钮，每个选项包含文本、跳转目标等子字段。编辑时支持按顺序排列、批量修改，输出为完整的脚本代码块。
